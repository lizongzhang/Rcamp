绘制下列向量的直方图,为直方图填充你喜爱的颜色。

2.1

2.1 生成一个服从均匀分布的向量，该向量包含1000个值。

set.seed(123) #设置随机数种子，保证每次运行代码生成的随机数相同
x <- runif(1000)
hist(x, breaks = 20,col = 5,
     main = "Histogram of U(0,1)",
     ylim = c(0,80),
     yaxt = "n"
     )


# 设置 Y 轴刻度
axis(side = 2, las = 1,at = seq(0, 80, 20), 
     labels = seq(0, 80, 20),font = 0.5)
