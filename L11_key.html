1.1 将数据集lbw中的变量名全部转换为小写字母，并将数据集lbw复制给对象df，将df中的列id, 列bwt删除.

load("lbw.rda") 
names(lbw) <- tolower(names(lbw))

1.2 对df中的变量进行转换：将ptl中等于2或3的组别，都合并到ptl等于1的组别中。将ftv中等于3，4，5，6的组别，都合并到ftv = 2的组别中。将low中等于”BWT <= 2500g”的组别，都转换为1，其余组别转换为0。

df$ptl <- ifelse(df$ptl == 2 | df$ptl == 3, 1, df$ptl) %>% 
  as.factor()

                   df$ftv == 5| df$ftv == 6, 
                 2, 
                 df$ftv) %>% 
    as.factor()

df$low <- ifelse(df$low == "BWT <= 2500g", 1, 0)

1.5 报告多元logistic回归模型，模型1的解释变量包括age, lwt, smoke, ui, ptl，ht, ftv; 模型2的解释变量包括age, lwt, smoke, ui, ptl，ht, ftv, 以及race. 对比这两个模型的估计结果，评估在模型1中，是否应该引入race？race是否可以视作模型1中的混杂因素？

model1 <- glm(low ~ age + lwt+ smoke + ui + ptl + ht + ftv,
          df, family = "binomial")


m1 <- tbl_regression(model1,
               exponentiate = TRUE,  #被解释变量odds ratio
               estimate_fun = ~ style_number(.x, digits = 3),

model2 <- glm(low ~ age + lwt+ smoke + ui + ptl + ht + ftv + race,
