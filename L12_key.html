2 习题2

数据文件：boot包中的melanoma

2.1 查看数据文件的帮助文档。

?boot::melanoma
library(boot)
data(melanoma)

2.2 将melanoma保存为对象df, 在df中追加变量的变量status_os。当status等于2时，代表个体是存活的，status_os等于0；若status等于1或3，代表个体死于黑色素瘤melanoma，或者死于其他疾病，令status_os等于1

df <- melanoma %>%
  mutate(status_os = if_else(status == 2, 0, 1))

2.3 使用2.2中创建的的数据框df ,建立Cox模型, 解释变量包括age，sex，thickness，ulcer. 问：能否在模型引入age或者thickness的rcs项，模型中的非线性关系是否显著？建立一个适当的含有rcs项的Cox模型。

library(rms)
dd<-datadist(df)
options(datadist='dd')
cox_rcs <- cph(Surv(time, status_os) ~ rcs(age, 4) + rcs(thickness, 4) +sex,

# 绘制分性别曲线
cox_rcs_pred_HR <- Predict(cox_rcs, 
                           thickness,
                           age = round(quantile(df$age, seq(0.1, 0.9, 0.2))),
                           sex = c(0, 1),
                           np = 100,
                           fun = exp,
                           ref.zero = TRUE) %>% 
  as.data.frame()

cox_rcs_pred_HR %>% 
  ggplot(aes(thickness, yhat, col = as.factor(sex))) +
  geom_point() +
  facet_wrap( ~ age) +
  geom_ribbon(aes(ymin = lower, ymax = upper), alpha = 0.2) +
  geom_hline(yintercept = 1, linetype = 3) +
  theme_bw() +
